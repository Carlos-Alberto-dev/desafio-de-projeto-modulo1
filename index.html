História da Web

A internet tem suas origens em um projeto de pesquisa financiado pelo governo dos Estados Unidos, conhecido como ARPANET (Advanced Research Projects Agency Network). ARPANET tinha como objetivo criar uma rede de comunicação robusta e descentralizada que pudesse resistir a falhas e ataques nucleares.

Em 1969, a primeira mensagem foi enviada pela ARPANET entre dois computadores localizados na Universidade da Califórnia, Los Angeles (UCLA) e no Stanford Research Institute (SRI). Este evento marcou o nascimento da internet como conhecemos hoje.

À medida que a ARPANET crescia, outras redes de comunicação começaram a se conectar a ela, formando uma rede interconectada de redes, o embrião da internet. Em 1983, a ARPANET adotou o protocolo TCP/IP (Transmission Control Protocol/Internet Protocol), que se tornou o padrão para comunicação na internet.

Nos anos seguintes, a internet cresceu exponencialmente com a participação de universidades, instituições de pesquisa e, eventualmente, o setor comercial e o público em geral.

Introdução sobre Cliente

É um software ou dispositivo que solicita e recebe informações de um servidor web. Temos exemplos de clientes web utilizados pelos usuários para acessar conteúdo online. O cliente web interpreta e exibe conteúdo web, como páginas HTML, CSS, JavaScript, imagens e vídeos, para os usuários. Os clientes web enviam solicitações para servidores web para obter dados ou recursos, como páginas da web, arquivos, serviços ou informações dinâmicas.

Introdução ao server

O servidor web é um software que armazena, processa e entrega conteúdo web em resposta a solicitações dos clientes web. Ele é responsável por hospedar e disponibilizar sites, aplicativos web e outros recursos online para os usuários. Exemplos populares de servidores web incluem o HTTP Server, Nginx, Microsoft Internet Information Services (IIS) e o servidor web integrado em muitas plataformas de desenvolvimento, como Django (Python) e Express.js (Node.js).

linguagens de programação

As linguagens de programação desempenham papéis cruciais tanto no lado do cliente quanto no lado do servidor em desenvolvimento web.

Linguagens lado cliente

  • HTML é a linguagem base da web e é usada para estruturar e organizar o conteúdo de uma página web.
  • CSS é usado para estilizar e formatar o conteúdo HTML, permitindo controlar aspectos visuais como cores, fontes, layout e animações.
  • JavaScript é uma linguagem de programação de alto nível que permite adicionar interatividade e dinamismo às páginas web.

Linguagens lado server

  • PHP é uma linguagem de script do lado do servidor amplamente utilizada para desenvolvimento web.
  • Python é uma linguagem de programação versátil que é frequentemente usada no desenvolvimento web.
  • Node.js é uma plataforma de desenvolvimento baseada em JavaScript que permite executar código JavaScript no lado do servidor.

Introdução ao HTML

Uma "tag" no contexto da web geralmente se refere a um elemento de marcação usado em linguagens como HTML (Hypertext Markup Language) para definir e estruturar o conteúdo de uma página web.

Algumas Tags utilizadas:

  • Tags <h1> à <h6> São utilizadas para título principal e subtítulos, sendo o H1 o principal.
  • Tag <p> define um paragráfo.
  • Tag <mark> destaca um texto.
  • Tag <small> define textos menores.
  • Tag <i> define o texto em itálico.
  • Tag <u> exibe um texto em sublinhado.
  • Tag <strong> exibe o texto em negrito.
  • Tag <ol> cria uma lista ordenada.
  • Tag <ul> cria uma lista desordenada.
  • Tag <li> cria ítens em uma lista.
  • Tag <a> é utilizada para criar links.
  • Tag <hr> é utilizada para criar uma linha na pagina web.
  • Tag <sub><sub> define texto subscrito.
  • Tag <sup> define texto sobrescrito.
  • Tag <blockquote> especifica uma seção citada de outra fonte.
